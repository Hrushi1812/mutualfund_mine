# Project Requirements

To set up the project locally, ensure you have Node.js installed. Then, follow these steps:

## 1. Install Dependencies

Run the following command in your terminal to install all required packages:

```bash
npm install
```

If you are setting up the project from scratch, use these commands to install the specific packages used:

### Core Framework
- `react`
- `react-dom`
- `react-router-dom`

### UI & Styling
- `tailwindcss`
- `lucide-react` (Icons)
- `framer-motion` (Animations)

### 3D & Advanced Effects (New)
- `@react-three/fiber` (3D Canvas)
- `@react-three/drei` (3D Helpers)
- `lenis` (Smooth Scrolling)

### Utilities
- `axios`
- `clsx` (Optional, for class merging)
- `tailwind-merge` (Optional, for class merging)

## 2. Installation Command

One-liner to install the new UI dependencies:

```bash
npm install @react-three/fiber @react-three/drei lenis framer-motion lucide-react
```

## 3. Dev Dependencies

- `vite`
- `eslint`
- `autoprefixer`
- `postcss`
